# About The Tests
## Location
* `test/`
## Contents
* `test_flight_parsing.py`
    * Test core parsing code.
* `test_heathrow_parsing`
    * Test heathrow specific parsing logic.

## How to Run
Go to folder above test folder:
```
cd test/
cd ../
```
To run all tests:
```
python -m unittest
```
To run a particular test:
```
python -m unittest test.test_module_name
```
